# Volkswagen GraphQL API

Volkswagen provides connected vehicle APIs through the We Connect platform. The API covers vehicle status, remote operations (lock, ventilation, charging), electric range and charging data, trip history, geofencing, and speed alerts.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Volkswagen REST API
**Documentation:** https://www.volkswagen.de/en/models/id-4/we-connect.html

- Reference: https://www.npmjs.com/package/we-connect-api
- Schema: https://github.com/api-evangelist/volkswagen
